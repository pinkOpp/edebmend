# Как сохранить файл в гит 
## Первая команда 
* git add (файл)
* Например: git add fio.c
копирует
## Вторая команда 
* git commit -m "Добавил программу fio.c"
* Добовляем комент 
## Третья команда 
* git push -u origin main
* окончательно сохраняет после первого раза можно использовать команду
* git push
