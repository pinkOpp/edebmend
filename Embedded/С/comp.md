# Как скомпилировать программу на C
## 1. Проверить, что мы находимся в нужной папке 
pwd
ls
Если в списке есть one.c, значит всё верно.
## 2. Если мы не в нужной папке
cd ~/Desktop/edebmend/Embedded/C/Day01_Basics
## 3. Скомпилировать программу
gcc one.c -o one
## 4. Запустить программу
./one
### 5. Сохранить изменения в GitHub
git status
git add .
git commit -m "Описание изменений"
git push
